# Notice

This repository contains two kinds of material.

## Course Sample Codes

The files inside `course_sample_codes/` are original course sample codes for the Data Structures and Algorithms course.

They are included in this repository with permission so they can be used as course references and compared with student practice implementations.

## Test Zone

The files inside `test_zone/` are my own practice implementations, test runners, and learning helpers.

These files are my own practice work and are part of the same MIT-licensed repository.

## Purpose

The purpose of this repository is educational. It is meant to help students read implementations, run simple tests, compare expected output with actual output, and understand common edge cases in data structures and algorithms.
